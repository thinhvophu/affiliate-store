## 1. Story & scope

**As a** site visitor, **I want to** read the full affiliate-disclosure statement at
`/cong-bo-tiep-thi-lien-ket/`, **so that** I trust that the site's monetization is transparent
and adds no cost to me.

Deliver a Vietnamese, statically pre-rendered (SSG) Affiliate Disclosure page:

- Single `<h1>`, semantic body, global `<Header />` / `<Footer />` chrome (already mounted in
  `app/layout.tsx` — no extra wiring).
- Lead paragraph renders `AFFILIATE_DISCLOSURE_VI` from `lib/disclosures.ts` **verbatim** (import
  the constant — no copy-pasted literal).
- Below the lead, a long-form Vietnamese explanation of the Shopee affiliate relationship and the
  "không phát sinh thêm chi phí cho bạn" (no extra cost) point.
- Unique Vietnamese `<title>`, `<meta name="description">`, canonical
  `${NEXT_PUBLIC_SITE_URL}/cong-bo-tiep-thi-lien-ket/`.
- Fixes the currently-broken footer link from US00022.

**Out of scope (per F0005 §2):** JSON-LD / Open Graph / breadcrumbs / sitemap inclusion (→ F0009);
cookie-consent banner; the `<AffiliateDisclosure>` component (→ US00051); the Privacy Policy page
(→ US00052). SEO baseline only.

## 4. Page structure

`page.tsx` (Server Component, no `"use client"`, no data loaders):

```
export const metadata: Metadata = {
  title: "Công bố tiếp thị liên kết | aff-store",            // unique Vietnamese title
  description: "<≤160-char Vietnamese summary of the affiliate disclosure>",
  alternates: {
    canonical: `${process.env.NEXT_PUBLIC_SITE_URL ?? ""}/cong-bo-tiep-thi-lien-ket/`,
  },
};

export default function AffiliateDisclosurePage() {
  return (
    <article className={styles.page}>
      <h1>Công bố tiếp thị liên kết</h1>
      <p className={styles.lead}>{AFFILIATE_DISCLOSURE_VI}</p>   {/* verbatim, imported */}
      <section> … long-form Vietnamese explanation … </section>
    </article>
  );
}
```

Long-form body (real Vietnamese copy, not lorem ipsum — §5 edge case) covers, with `<h2>`
sub-sections under the single `<h1>`:

1. Quan hệ tiếp thị liên kết với Shopee — links open Shopee via affiliate URLs; we may earn a
   commission.
2. "Không phát sinh thêm chi phí cho bạn" — the commission is paid by Shopee, the buyer's price is
   unchanged.
3. Tính minh bạch / cam kết biên tập — recommendations are content-driven, not pay-for-placement.
4. (Optional) pointer to the Privacy Policy page once US00052 ships.

No full-width shell needed — render inside a centered prose container (consistent with the
detail-page "no `<ShellLayout />`" precedent); Header/Footer come from the root layout.
